NVIDIA 3080 FE

Simple website that will check at a 15s interval if the stock of the 3080 FE changed.

It fetches the same json file that the website uses to display their 'Out of stock/Add to cart'. If the status change in any sort of way, this website will send you an alert and push notification if you allow it.

Leave this page running in the background to get a notification. When you get a notification, clicking on it will open up NVIDIA's store, ready for you to click on that nice add to cart button.
